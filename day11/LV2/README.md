## 📚 Day 11

### 🚀 Level 2

#### 🤔 문제 설명

- 숫자 배열이 있습니다.
- 당신의 임무는 홀수 번째(1 base index) 숫자를 오름차순으로 정렬하는 것이지만 짝수는 그 자리에 있어야합니다.
- 0은 홀수가 아니므로 이동할 필요가 없습니다. 
- 빈 배열일 경우 빈 배열을 반환해야합니다.

#### 🎯 TestCase

```javascript
sortArray([5, 3, 2, 8, 1, 4]) // [1, 3, 2, 8, 5, 4]

sortArray([5, 3, 2, 8, 1, 4]) // [1, 3, 2, 8, 5, 4] 

sortArray([5, 3, 1, 8, 0])    // [1, 3, 5, 8, 0]
```
